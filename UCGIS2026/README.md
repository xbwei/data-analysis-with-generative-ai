# UCGIS 2026 Materials

This folder contains materials for Dr. Xuebin Wei's UCGIS 2026 workshop and lightning talk.

## Workshop

### Advancing Social Media Analytics in Neo4j: Multimodal Embeddings and AI Agents with Knowledge Graphs

The live workshop focuses on an end-to-end text-based workflow:

1. Create a free Neo4j AuraDB instance.
2. Load synthetic social media data into a knowledge graph.
3. Explore the graph with Neo4j AI Query.
4. Create Gemini text embeddings and a Neo4j vector index.
5. Run GraphRAG, Cypher-Augmented Generation, and Geo-Augmented GraphRAG in Python.
6. Build a free no-code Neo4j Graph Agent.

The official workshop title includes multimodal embeddings. Text-image embeddings are provided as an optional follow-up tutorial rather than a live exercise.
